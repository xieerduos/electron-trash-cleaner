# Electron 垃圾清理器

Electron 垃圾清理器是一款基于 Electron 和 Node.js 构建的桌面应用程序。它利用子进程和 Worker 线程的能力，有效地清理和管理你的系统垃圾，提高你的计算机的整体性能和效率。

## 安装

克隆仓库：

```bash
git clone git@github.com:xieerduos/electron-trash-cleaner.git
```

## 导航到项目目录：

```bash
cd electron-trash-cleaner
```

## 安装依赖：

```bash
npm install
```

## 启动应用程序：

```bash
npm start
```
